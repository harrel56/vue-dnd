# vue-dnd

Simple vue 3.x drag and drop plugin

## Example
```html
<DropZone>
  <div class="z1">
    <Draggable><div class="d1"></div></Draggable>
    <Draggable><div class="d2"></div></Draggable>
    <Draggable><div class="d3"></div></Draggable>
  </div>
</DropZone>
<DropZone dropping-class="test-dropping">
  <div class="z2">
    <Draggable><div class="d1"></div></Draggable>
    <Draggable><div class="d2"></div></Draggable>
  </div>
</DropZone>
```
